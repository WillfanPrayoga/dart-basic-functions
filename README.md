📱 Dart Basic Functions
Project ini merupakan implementasi dasar fungsi (functions) dalam bahasa Dart.
Program dirancang modular dengan beberapa fitur sederhana untuk melatih logika dan struktur kode.
🚀 Features
👋 Fungsi menyapa pengguna
🔲 Kalkulator luas persegi
💸 Kalkulator diskon harga
🧩 Struktur kode modular (berbasis fungsi)
🛠️ Tech Stack
Language: Dart
Tools: VS Code / Terminal

⚙️ How It Works
Program berjalan melalui fungsi utama:
void main()

🔁 Execution Flow
Program dimulai dari main()
Memanggil fungsi sapa()
Menghitung luas persegi dengan hitungLuasPersegi()
Menghitung diskon dengan hitungDiskon()
Menampilkan hasil ke console

🧩 Functions Breakdown
1. 👋 Greeting Function
void sapa(String nama)
Menampilkan pesan sapaan ke user
Menggunakan string interpolation
2. 🔲 Square Area Calculator
int hitungLuasPersegi(int sisi)
Menghitung luas persegi
Rumus: sisi × sisi
Return: nilai integer

4. 💸 Discount Calculator
double hitungDiskon(double harga, double persenDiskon)
Menghitung harga setelah diskon
Rumus:
harga - (harga × persen / 100)
Return: nilai double

▶️ Getting Started
1. Clone Repository
git clone https://github.com/WillfanPrayoga/dart-basic-functions.git
cd dart-basic-functions
2. Run Program
dart run main.dart
